# 🎨 Art Shine

Art Shine is a full-stack MERN marketplace platform where users can explore, buy, and sell creative artworks online. The platform includes a modern user website, seller dashboard, real-time chat system, secure authentication, and order management features.

---

# 🚀 Features

## 👤 User Side
- User Authentication (Login / Register)
- Browse Art Products
- Product Details Page
- Add to Cart
- Place Orders
- Real-Time Chat with Sellers
- Responsive Modern UI
- Smooth Animations using GSAP & Framer Motion

## 🛍 Seller Dashboard
- Seller Authentication
- Add New Products
- Manage Products
- Manage Orders
- Chat with Customers
- Upload Product Images

## ⚡ Backend Features
- REST API with Express.js
- MongoDB Database Integration
- JWT Authentication
- Image Upload using Multer
- Real-Time Communication using Socket.IO
- Email Functionality using Nodemailer

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone <YOUR_GITHUB_REPO_LINK>
```

---

## 2️⃣ Install Backend Dependencies

```bash
cd Server
npm install
```

---

## 3️⃣ Install Frontend Dependencies

### User Frontend

```bash
cd artshine
npm install
```

### Seller Dashboard

```bash
cd seller
npm install
```

---

# ▶️ Run Project

## Start Backend

```bash
cd Server
npm start
```

## Start User Frontend

```bash
cd artshine
npm run dev
```

## Start Seller Dashboard

```bash
cd seller
npm run dev
```

---

# 🔐 Environment Variables

Create `.env` files inside:

- `Server`
- `artshine`
- `seller`

Example:

```env
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
EMAIL=your_email
PASSWORD=your_email_password
```
